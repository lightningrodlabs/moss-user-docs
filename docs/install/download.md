# Download Moss

Moss is a desktop application for **macOS**, **Windows**, and **Linux**.

## Latest stable release

Download the latest public release from **[moss.social](https://moss.social)**.

The download page has installers for:

- macOS (Apple Silicon and Intel)
- Windows
- Linux

## Unstable / development builds

If you want to try an in-development build (for example to test a feature that hasn't landed in a release yet), you can pick one from the [GitHub releases page](https://github.com/lightningrodlabs/moss/releases).

Unstable builds may contain bugs and breaking changes. They are not recommended for groups whose data matters to them.

## System requirements

- A reasonably recent 64-bit machine
- Enough disk space for Moss, its conductor, and the data your group accumulates (start with ~1 GB free; most groups will use far less)
