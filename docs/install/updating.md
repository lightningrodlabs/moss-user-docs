# Updating Moss

Moss checks for updates and will prompt you when a new stable release is available. You can also update manually by downloading the latest version from [moss.social](https://moss.social).

## Release notes

Each release is published with release notes on the [GitHub releases page](https://github.com/lightningrodlabs/moss/releases). Notes call out:

- New features
- Bug fixes
- Any breaking changes (these are rare but do happen during the alpha period)

## Compatibility with other group members

Moss updates sometimes include changes to the underlying Holochain version or to how groups communicate. When that happens, **all members of a group generally need to update** to stay in sync.

If you find your group can't see each other's activity after an update, the most common cause is that some members haven't updated yet. Check the release notes for the version you're running.

## Tools update separately

Tools have their own versions and their own release cadence. See [Adding Tools](../groups-and-tools/adding-tools) for how tool updates are handled within a group.
