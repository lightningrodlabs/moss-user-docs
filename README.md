# Moss User Docs

End-user documentation for [Moss](https://github.com/lightningrodlabs/moss), published at [docs.moss.social](https://docs.moss.social).

Built with [VitePress](https://vitepress.dev/). Deployed via GitHub Pages from `main`.

## Local development

```bash
yarn install
yarn docs:dev
```

## Build

```bash
yarn docs:build
yarn docs:preview
```

## Structure

- `docs/intro/` — background, intent, use cases
- `docs/install/` — downloading and first launch
- `docs/groups-and-tools/` — creating groups, inviting members, adding tools
- `docs/public/` — static assets (screenshots, logos, `CNAME`)
